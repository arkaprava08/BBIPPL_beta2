J2EE

Overview

J2EE technology evolves through community process and is a very popular open source platform. It provides standards and mechanisms for the 3 basic layers of a typical web application - presentation layer, business logic layer and data access layer. It provides for asynchronous communication, mail API, and security as well. It tries to restrict itself to provide standards and leave it to tool providers to implement these standards. The tool provider then customizes it for the end developer. This greatly simplifies the developer's job and allows customer to choose the tool, which suits them most.

Brick and Byte monitors this continually evolving technology through the J2EE focus group. The group also evaluates the new tools coming to market from different vendors to implement the technology. It provides support to the project teams during the architecture and design phase.

PHP

Overview

PHP, which stands for "PHP: Hypertext Preprocessor" is a widely-used Open Source general-purpose scripting language that is especially suited for Web development and can be embedded into HTML. Recently it is gaining lot of popularity in small to medium size web applications and making inroads in large size applications.

PHP can be used on all major operating systems, including Linux, many UNIX variants, Microsoft Windows, Mac OS X, etc. PHP has also support for most of the web servers today. With PHP 5 it has become largely object oriented. It does support all major databases and widely used protocols.

Considering this, we have formed a PHP focus group which keeps itself abreast the latest in PHP and evaluates various open source extensions and frameworks available in PHP.
